# BMI Calculator

เครื่องคำนวณค่าดัชนีมวลกาย (BMI) ที่สวยงามและใช้งานง่าย

## 🌐 Demo
🔗 **Live Demo**: [https://bmi-calculator.vercel.app](https://bmi-calculator.vercel.app)

## ✨ คุณสมบัติ
- 🎨 ดีไซน์สวยงามแบบเด็กรุ่นใหม่
- 📱 Responsive Design
- 🌈 Color-coded ผลลัพธ์
- 🖼️ รูปคำเตือนเมื่อน้ำหนักเกิน
- ⚡ Animation ที่นุ่มนวล

## 🎯 การใช้งาน
1. กรอกน้ำหนัก (กิโลกรัม)
2. กรอกส่วนสูง (เซนติเมตร)
3. กดปุ่ม "คำนวณ BMI"
4. ดูผลลัพธ์และคำแนะนำ

## 📊 เกณฑ์การแปลผล BMI
- **< 18.5** - น้ำหนักน้อย (สีน้ำเงิน)
- **18.5-24.9** - น้ำหนักปกติ (สีเขียว)
- **25.0-29.9** - น้ำหนักเกิน (สีเหลือง) 🖼️
- **≥ 30.0** - อ้วน (สีแดง) 🖼️
